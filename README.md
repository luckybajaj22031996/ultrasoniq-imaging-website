# UltraSoniq Imaging & Interventions

Website for UltraSoniq Imaging & Interventions — a specialised sports injury imaging & pain clinic in Bhopal, led by Dr. Suvinay Saxena and Dr. Satya Jha.

## Live site
https://ultrasoniq-imaging.netlify.app

## Stack
- Single-page static HTML with embedded CSS and JS
- Deployed on Netlify via continuous deployment from this repo

## File structure
- `index.html` — the website
- `img1.png` — favicon
- `img2.png` — nav logo
- `img3.jpg` — Dr. Suvinay Saxena photo
- `img4.jpg` — Dr. Satya Jha photo

## Making changes
1. Edit `index.html`
2. Commit and push to `main` — Netlify auto-deploys

## Local preview
Just open `index.html` in a browser, or run:
```
python3 -m http.server 8000
```
Then visit http://localhost:8000
